# Azure Observability

Monitor these signals:

- Application Gateway WAF blocks, 5xx, latency, and backend health.
- Function failures, duration, cold starts, and dependency errors.
- AKS pod restarts, node pressure, and HPA saturation.
- SQL DTU/vCore usage, blocking, storage, and failover status.
- Service Bus active messages, dead-letter messages, and age.
- Event Hubs incoming events, throttles, and Capture failures.

Use Azure Monitor, Application Insights, and Log Analytics workbooks.
